Sobre E-tech SYS

Ferramenta desenvolvida para facilitar o trabalho de funcionários na empresa E-tech Informática TELECOM.

Lançamento 03/11/2023

Atualização 07/11/2023:

Implementado melhorias na responsividade, suportando telas com resolução a partir de 300x400.

Atualização 25/11/2023:

Corrigido os valores com uma quantidade indesejada de casas decimais, valores gerados agora são arredondados automáticamente.
Valores com casas decimais menor que R$0.50 serão zeradas e as maiores serão arredondadas para o valor inteiro, ou seja, R$0.49 se tornará R$0.00 e R$0.51 se tornará R$1.00.
